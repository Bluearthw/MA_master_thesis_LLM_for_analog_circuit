\begin{abstract}
Analog circuit design automation seems to step into next generation due the benefit of large-language model (LLM). Some systems still require manual functional netlist as input to start automatic sizing. To make the workflow more automatic, this dissertation introduces a multi-agent system (MAS) which take 3 types of malfunctioning circuit, transform them into functional circuits for simulator and send it to an reinforcement learning (RL) sizer to look for possible solution targeting certain specifications. It has netlist generator which generate the first netlist draft and error fixer to fix the error in the netlist if exists. It can identify 3 types of circuit from a dataset and make them a simulatable netlist for certain specifications. Then, the following RL sizer takes the netlist and outputs optimized solution.
